# git

A ry library for git operations.

## Installation

```bash
ry --install git
```

## Usage

```bash
ry git test        # Test the library
ry git version     # Show version
```

## Development

To test locally:
```bash
ry docs/libraries/git/git.yaml test
```

## Commands

- `test`: Verify library is working
- `version`: Display library version
