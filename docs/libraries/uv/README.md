# uv

A ry library for uv operations.

## Installation

```bash
ry --install uv
```

## Usage

```bash
ry uv test        # Test the library
ry uv version     # Show version
```

## Development

To test locally:
```bash
ry docs/libraries/uv/uv.yaml test
```

## Commands

- `test`: Verify library is working
- `version`: Display library version
